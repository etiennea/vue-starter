# Vue starter

> Personnal Vue starter

## Features

* Webpack 4
* Vue class component with Babel 7 or TypeScript
* Sass enabled by default
* PWA & Workbox plugins
* Vue plugins
  * vue-i18n
  * vue-meta
  * vuelidate
* Webpack plugins
  * PostCSS with Autoprefixer & CSSNext
  * Imagemin
  * Critical CSS
  * Dotenv integration
* Units tests with Mocha & @vue/test-utils
* E2E tests with Cypress
